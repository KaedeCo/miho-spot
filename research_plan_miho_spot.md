# Miho-spot 舆情监测系统 - 研究与开发计划

## 项目概述
开发一个基于多平台热搜爬取的米哈游舆情检测系统，集成网络爬取、舆情监测、数据分析为一体。

## 技术架构

### 前端技术栈
- React 18 + TypeScript + Vite
- TDesign React (UI组件库)
- Tailwind CSS (样式)
- Recharts / ECharts (数据可视化)
- React Router (路由)

### 后端技术栈
- Python 3.10+
- FastAPI (REST API)
- PyQt6 (桌面GUI)
- SQLite (数据持久化)
- Playwright/Selenium (网页爬取)
- jieba + SnowNLP (中文情感分析)

## 关键功能模块

### 1. 用户认证模块
- 知乎账号登录
- 抖音账号登录
- 贴吧账号登录
- Cookie管理与会话保持

### 2. 热榜爬取模块
- 知乎热榜爬取
- 抖音热榜爬取
- 贴吧热榜爬取
- 定时任务调度

### 3. 关键词过滤模块
- 二游圈关键词词典（可扩展）
- 米哈游相关：原神、崩坏：星穹铁道、崩坏3、绝区零、米哈游、miHoYo、大伟哥等
- 竞品相关：明日方舟、鸣潮、无限暖暖、幻塔等
- CV相关：常见二次元CV名字
- 前端关键词管理页面

### 4. 情感分析模块
- 三级分类：Positive/Negative/Irrelevant
- 对Irrelevant内容判断相关游戏
- 深度分析：爬取热搜下100条帖子/视频
- 帖子级情感三分：Positive/Negative/Neutral

### 5. 数据可视化模块
- 饼状图（情感分布）
- 条形图（情感分布）
- 图表切换控件
- 跨日统计（7天/30天/自定义）

### 6. 持久化存储
- SQLite数据库
- 热搜数据表
- 帖子/视频数据表
- 情感分析结果表
- 每日统计汇总表
